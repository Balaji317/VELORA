# 👗 VELORA

A modern luxury fashion website built using **HTML5** and **CSS3**. VELORA features a clean, elegant design with responsive layouts, modern typography, and smooth user interactions.

## 🌐 Live Demo

👉 https://balaji317.github.io/VELORA/

---

## 📖 About

VELORA is a front-end fashion website created to practice modern web development fundamentals. The project focuses on responsive layouts, semantic HTML, Flexbox, and clean UI design.

---

## ✨ Features

- Modern luxury fashion landing page
- Responsive design
- Offer section with background image
- Editor's Picks section
- New Arrivals gallery
- Smooth hover effects
- Professional footer
- Semantic HTML5 structure
- Organized CSS
